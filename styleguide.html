Typography

All headings for both Helvetica Neue font stack and Garamond font stack are available, colors are optional.

The default type color for Helvetica Neue is a dark gray, HEX number, #2d2d2d, RGB color, 45, 45, 45 and CMYK color, 70, 64, 63, 64.
